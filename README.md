# SmartSchool Frontend – Sistema de Gestión de Eventos Escolares

![Evento escolar](./Evento%20escolar%20al%20aire%20libre.png)

---

## 📌 Descripción

El **Frontend de SmartSchool** corresponde a la interfaz web del Sistema de Gestión de Eventos Escolares. Esta aplicación permite a estudiantes, docentes, padres de familia y personal administrativo interactuar con la plataforma para consultar, organizar y gestionar eventos dentro de una institución educativa.

La interfaz está diseñada para ser **intuitiva, rápida y accesible**, permitiendo el acceso desde diferentes dispositivos como computadores, tablets o teléfonos móviles.

---

## 🎯 Objetivo

Desarrollar una interfaz web moderna que permita a los usuarios visualizar, registrar y gestionar eventos escolares mediante una plataforma centralizada.

---

## ⚙️ Funcionalidades Principales

El frontend permite realizar las siguientes acciones:

- Inicio de sesión de usuarios.
- Visualización de eventos escolares.
- Registro y gestión de eventos.
- Gestión de usuarios.
- Visualización del calendario de eventos.
- Recepción de notificaciones.
- Consulta de reportes.

---

## 🛠️ Tecnologías Utilizadas

- **React**
- **Vite**
- **JavaScript**
- **HTML5**
- **CSS3**
- **Axios (consumo de API)**

---

## 🔗 Integración con Backend

El frontend consume una **API REST desarrollada en Laravel**, permitiendo gestionar:

- Usuarios
- Eventos
- Recursos
- Notificaciones
- Reportes
